앱 이름: 유학생들을 위한 대학생활 가이드

문제 정의하기 (Refining Your Problem Statement)

유학생들은 한국 대학의 학사제도(학점, 졸업요건, 수강신청 규칙 등)를 한 번에 이해하기 어렵고, 관련 정보가 여러 페이지에 분산되어 있다.
또한 수강신청 방법과 일정, 제한 조건이 복잡하지만 이를 쉽게 설명해주는 통합 가이드가 부족하다.

특히 한국어 중심의 안내와 홈페이지 번역 오류로 인해, 유학생들은 잘못된 페이지로 이동하거나 정보를 정확히 이해하지 못하는 문제가 발생한다.
이로 인해 중요한 학사 일정(수강신청, 정정 기간, 시험 등)을 놓치거나 학업 수행에 어려움을 겪는다.
👉 따라서 본 웹앱은
학사 정보를 한 곳에 통합하고, 번역 오류 없이 쉽고 정확하게 이해할 수 있도록 돕는 서비스를 목표로 한다.




사용자 정의하기 (Understanding the User)

예상 사용자

한국 대학에 재학 중인 유학생 (교환학생, 학부/대학원)
한국 대학 시스템이 익숙하지 않은 신입생
수강신청 및 학사제도에 어려움을 겪는 재학생



사용 상황

수강신청 전에 신청 방법과 규칙을 빠르게 확인할 때
졸업요건 및 이수학점을 정리해서 이해하고 싶을 때
시험 전에 과목별 평가 방식을 미리 확인할 때
학사 일정을 놓치지 않기 위해 확인할 때
한국어 공지를 이해하기 어려워 쉬운 설명이 필요할 때
홈페이지 번역 오류로 원하는 정보를 찾지 못했을 때

① 사용자가 웹사이트 접속
② 학교 및 언어 설정 (최초 1회 저장)
③ 메인 화면에서 주요 기능 선택
④ 학사제도 → 쉬운 설명 확인
⑤ 수강신청 가이드 → 단계별 안내 확인
⑥ 과목 검색 → 평가 방식 및 후기 확인
⑦ 공지 클릭 → 번역 + 쉬운 설명 확인
⑧ 학사 일정 확인 → 알림 설정
⑨ 즐겨찾기 저장 → 빠른 접근
