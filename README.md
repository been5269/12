# UniGuide Global (prototype)

정적 웹앱(HTML/CSS/JS) 형태의 프로토타입입니다.

## 실행

- `index.html`을 브라우저로 열면 됩니다.
- 로컬에서 간단한 서버를 띄우고 싶다면(권장):
  - VSCode/Cursor의 Live Server 확장
  - 또는 Python: `python -m http.server`

## 코드 주입(사용자 제공 코드)

- `index.html`의 `#slot-guide-content`, `#slot-mypage-content`
- `src/app.js`의 `renderTaskList()` (개발계획서 기반 단계/체크리스트 렌더링 자리)
